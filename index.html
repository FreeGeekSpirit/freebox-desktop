    Freebox Desktop

    Freebox Desktop est un ensemble d'application native permettant de gerer votre freebox. Basé sur l'API 1.0, elle comprend les composants suivant :

    • File System
    • Torrent

    Obtenir le code source

    Freebox Desktop est écrit en C++ / Qt5 et supporte Windows, MacOS et Linux. Vous pouvez obtenir le code source directement depuis github : 

    
$ git clone git@github.com:dridk/mafreebox.git
$ cd mafreebox
$ qmake mafreebox.pro 
$ make 
$ ./mafreebox

    Pour compiler , il vous suffit de télécharger Qt 5.1 Puis d'ouvrir le fichier mafreebox.pro puis de cliquer sur "Compiler". L'application est actuellement en phase de développement. Des binaires seront disponnible à la premiere version stable
